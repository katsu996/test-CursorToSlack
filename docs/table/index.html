docs/table/ の生成物

このページは GitHub Pages で配信される静的ファイルです。 難易度表の JSON は CI（tools/table-filter）がビルド時に書き出します。 リポジトリを clone した直後は .gitignore により次のファイルが無いことがあります。

役割の分担

この table/index.html は、生成される JSON への直リンクと beatoraja 登録時の注意だけを短くまとめた案内ページです。 リポジトリ全体のドキュメント索引・運用手順は docs/README.md （GitHub 上では Raw かリポジトリビューで開いてください）を正とします。

生成される主なファイル（ビルド後）

  • browser_rows.json — トップ表（docs/index.html）用
  • level_stats.json — level-stats.html 用
  • filtered_data.json / filtered_data_enriched.json — beatoraja 向けデータ
  • filtered_header.json — beatoraja 向けヘッダー（難易度表 URL にこの JSON を指定）

上記リンクは、デプロイ済みの Pagesではビルド後に有効です。ローカルで未生成のときは 404 になります。

beatoraja に難易度表を登録するとき

  1. 公開先のベース URL（例: https://<user>.github.io/<repo>/）を決める。
  2. 難易度表のヘッダー URL には、配信された table/filtered_header.json の絶対 URLを指定する。
  3. ヘッダー内の data_url が相対（ファイル名のみ）のときは、ヘッダーと同じディレクトリから filtered_data.json（または enriched）が取れる必要がある。

形式の詳細は docs/beatoraja-jbmstable-table-json.md および docs/beatoraja-difficulty-table-url-and-filtered-publish.md を参照してください。
