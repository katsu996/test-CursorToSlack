難易度表（songdata 突合・一覧）

Actions で songdata.db の条件（現在: 等速 BPM = minbpm = maxbpm）に合う行だけに絞った難易度表と、ローカル DB の song 列を表示します。 beatoraja の Table URL は、推奨どおり …/table/filtered_header.json の HTTPS URL を登録してください。 サイトのトップ（…/ または …/index.html）だけを登録する場合も、本ページの bmstable メタから同じヘッダーを解決できます。

列の表示切替（1 列ごとにチェック）。初期状態は従来どおり表示していた列のみオンです。追加の列にチェックを入れると表に出ます。

レベル別の曲数の集計は level-stats.html（table/level_stats.json）を参照してください。 手動での運用（DB・SQL・URL・push など）はリポジトリ直下の README.md。Actions の裏側は docs/github-actions-songdata-table-filter.md を参照。
